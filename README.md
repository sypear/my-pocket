# My Pocket
> React 연습을 위한 가계부 프로젝트입니다.<br/>
> 선택한 연도 별도 별로 수입/지출 내용을 확인할 수 있고, 월별 지출 차트를 통해 해당 연도의 월별 지출 현황을 확인할 수 있습니다.
